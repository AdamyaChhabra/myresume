# My Resume

This repository contains my personal resume created using basic HTML.

The purpose of this project is to present my academic details, skills, and other information in a structured and simple web format.

## Technologies Used
- HTML

## Features
- Profile photo
- Basic personal details
- Educational qualifications
- Skills and achievements
- Languages and hobbies

## How to View
Open the `resume.html` file in any modern web browser to view the resume.
